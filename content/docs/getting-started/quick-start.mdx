  ## Installation using Executable Package: Development Workflow

    Follow these steps to generate SQLPage assets and start the development server:

    <Tabs items={['SQLite', 'PostgreSQL']}>
      <Tab value="SQLite">
        ### SQLite Setup
        For SQLite DB give the below content in Spryfile.md
        ``` text
        ---
        sqlpage-conf:
          allow_exec: true
          port: 9277
          database_url: "sqlite://spry.sqlite.db?mode=rwc"
          web_root: "./dev-src.auto"
        ---
        ```
        <Steps>
          <Step key="gene-sql">
            ### Generate SQLPage Source Files
            
            Generate the SQLPage source files from your Spryfile.md:
            
            ```bash
            spry-sqlpage spc --fs dev-src.auto --destroy-first --conf sqlpage/sqlpage.json
            ```
          </Step>

          <Step key="start-dev">
            ### Start Development Server
            
            Run Spry in watch mode for live reloading:
            
            ```bash
            spry-sqlpage spc --fs dev-src.auto --destroy-first --conf sqlpage/sqlpage.json --watch
            ```
            
            <Callout type="info" title="Auto-Reload">
              The `--watch` flag automatically regenerates files when your Spryfile.md changes!
            </Callout>
          </Step>

          <Step key="dep-db">
            ### Deploy to Database
            
            Deploy SQLPage components to your SQLite database:
            
            ```bash
            spry-sqlpage task deploy
            ```
          </Step>

          <Step key="st-ser">
            ### Start SQLPage Server
            
            Launch the SQLPage server:
            
            ```bash
            SQLPAGE_SITE_PREFIX="" sqlpage
            ```
            
            Your application will be available at `http://localhost:8080`
          </Step>
        </Steps>
      </Tab>
      <Tab value="PostgreSQL">
        ### PostgreSQL Setup
        For PostgreSQL DB give the below content in Spryfile.md
        ``` text
        ---
        sqlpage-conf:
          allow_exec: true
          database_url: "postgresql://user:password@localhost:5432/mydb"
          web_root: "./dev-src.auto"
        ---
        ```

## Your First Spryfile

The `Spryfile.md` is where you define your application. Here's a simple example:

````markdown
# My First Spry App

## Database Setup

```bash prepare-db --descr "Initialize database"
sqlite3 app.db < schema.sql
```

## Home Page

```sql index.sql
SELECT 'text' AS component,
       'Welcome to Spry!' AS title,
       'This is my first SQLPage app' AS contents;
```
````
